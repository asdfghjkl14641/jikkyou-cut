# jikkyou-cut

ゲーム実況・配信切り抜きに特化した、完全無料・オープンソースの動画編集ツール。

長時間アーカイブから「文字起こしテキストを消すだけで動画もカットされる」テキストベース編集で、切り抜き作業を高速化します。

## なぜ作るか

| ツール | 課題 |
|---|---|
| Vrew | 汎用、従量課金、長時間動画でコスト爆発 |
| LosslessCut | テキスト編集なし・AI機能なし |
| DaVinci / Premiere | プロ向け、習得コスト高 |

**実況特化 × 無制限 × OSS** のポジションを取ります。

## MVP 機能

- 動画ファイル読み込み + プレビュー再生
- FFmpeg 内蔵 Whisper で日本語音声を自動文字起こし
- テキストベース編集 UI(テキスト削除 → 該当区間も削除)
- シンプルなタイムライン表示
- FFmpeg による最終動画書き出し

## 必要環境

- Windows 11
- FFmpeg 8.1 以降(`--enable-whisper` ビルド)を PATH に通すこと
- Node.js 22 以降
- Whisper モデルファイル(`ggml-base.bin` 等)を別途ダウンロードし、設定で絶対パス指定

## セットアップ

```bash
npm install
npm run dev
```

## MVP の制約

- 動画とプロジェクトファイル(`<basename>.jcut.json`)は同じディレクトリに配置されます。
- 編集の粒度は ASR(Gemini)が出力するキューに依存します。**1 キューの一部だけを残す/消す細かい編集はできません**。粒度を細かくしたい場合は ASR のセグメント長設定を調整するか、後段のフレーム単位編集機能(MVP 後)を待ってください。
